# IncidentSuite

This repository contains a modular automation suite for incident response and compliance recovery, built using Python and Jenkins.

## Features

- Self-healing scripts for service recovery
- Compliance check automation
- Jenkins integration for CI/CD
- Modular design for enterprise scalability

## Setup

1. Clone the repo
2. Install dependencies
3. Run `app.py` or trigger via Jenkins
